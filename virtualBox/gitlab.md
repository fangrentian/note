### 修改访问gitlab的url, 配置 `external_url`参数
```shell
[root@localhost ~]# vim /etc/gitlab/gitlab.rb
```

### 使配置生效

```shell
[root@localhost ~]# gitlab-ctl reconfigure
```

## 安装时指定url

```shell
[root@localhost ~]# EXTERNAL_URL="http://192.168.4.180" yum install -y gitlab-ee
```
